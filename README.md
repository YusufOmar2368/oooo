# oooo
This project builds machine learning models to classify breast tumors as malignant or benign using the Breast Cancer Wisconsin dataset from scikit-learn. The program loads the dataset, splits it into training and testing sets, and compares three classification models: Logistic Regression, Random Forest, and Support Vector Machine. Each model is trained on the training data and evaluated using metrics such as accuracy, precision, recall, F1 score, ROC-AUC, confusion matrices, and cross-validation to compare performance.

The implementation uses scikit-learn classes and pipelines to simplify preprocessing and model training. Important attributes include the feature data (X), the target labels (y), and metadata such as feature_names and target_names. Methods such as train_test_split(), fit(), predict(), and cross_validate() are used to train models, generate predictions, and evaluate performance. A limitation of the project is that it relies on a single dataset with limited samples and does not include extensive hyperparameter tuning, so results may not generalize to all medical datasets.
